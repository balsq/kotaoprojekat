Projektovanje šeme automatizovanog kotla (AutoCAD & EPLAN)
U okviru laboratorijskih vežbi na predmetu automatizacije, radio sam na izradi kompletne šeme automatizovanog kotla koristeći AutoCAD i EPLAN. U AutoCAD‑u sam izradio uopštenu tehničku šemu sistema, uključujući raspored glavnih komponenti i funkcionalnih celina. U EPLAN‑u sam zatim projektovao osnovnu električnu šemu koja obuhvata napajanje, upravljački deo i povezivanje senzora i izvršnih elemenata. Projekat je obuhvatio razumevanje procesa automatizacije, interpretaciju tehničke dokumentacije i praktičnu primenu alata za elektro‑projektovanje.
Automated Boiler System Schematic Design (AutoCAD & EPLAN)
As part of laboratory work in automation courses, I developed a complete schematic of an automated boiler system using AutoCAD and EPLAN. In AutoCAD, I created the general technical layout of the system, including the arrangement of key components and functional modules. Using EPLAN, I designed the basic electrical schematic, covering power distribution, control circuitry, and sensor/actuator connections. The project required understanding automation processes, interpreting technical documentation, and applying industry‑standard CAD tools for electrical design.
